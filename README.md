# Smart House Rental Management System

A full-stack web application developed using Python Flask and MySQL for managing rental properties efficiently.

## Features

* User Registration and Login
* Property Listing Management
* Property Image Upload
* Property Search
* Rent Prediction using Machine Learning
* Property Ratings and Reviews
* Property Booking System
* Property Enquiry System
* Analytics Dashboard
* PDF Report Generation
* Admin Dashboard

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* Machine Learning (Scikit-learn)
* Pandas
* Matplotlib
